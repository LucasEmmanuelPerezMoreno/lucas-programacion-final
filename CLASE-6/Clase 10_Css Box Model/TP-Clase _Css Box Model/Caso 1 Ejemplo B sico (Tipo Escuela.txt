Caso 1: Ejemplo Básico (Tipo Escuela)
Paso 1: Crear la Estructura HTML
Crea un archivo HTML con la estructura básica y dale un nombre con la extensión adecuada.
Paso 2: Crear el Archivo CSS
1.	Crea un archivo CSS y dale un nombre .css
2.	En el archivo anterior:
1.	Dentro de la clase box, añade margin para definir el espacio exterior del elemento.
2.	Dentro de la clase box añade padding para definir el espacio interior del elemento.
3.	Dentro de la clase box añade border para definir el borde del elemento.
4.	Dentro de la clase box añade box-sizing para incluir padding y borde en el tamaño total del elemento.
